# PRD：MBTI Learn（H5）— 场景对话英语游戏化学习（6–12岁亲子）

## 1. 背景与目标

### 1.1 背景
内容来源为“美国家庭万用英语表达”，已整理在 `part2.json`。产品希望以**家庭日常小场景**为载体，用游戏方式帮助小学生把句子**用在对话里**（而不是背单句）。

### 1.2 产品目标（MVP）
- **目标人群**：6–12 岁小学生；以及小学生亲子家庭共同使用。
- **端形态**：H5。
- **练习能力范围**：MVP 先做 **点读 + 选择**（不强依赖语音评测）。
- **学习目标**：能在对话情境中选出/组合出**合适的英文回应**，形成“情境→表达”的条件反射。
- **个性化**：用“MBTI + 学习性格/偏好”轻量分型（趣味测，不宣称严肃心理测量准确性），驱动玩法推荐与反馈风格。
- **商业化**：一开始订阅；提供**创始会员 0 元加入**（权益与转付费路径在 8 章）。

### 1.3 非目标（MVP 不做）
- 不做严肃量表（如完整 MBTI 量表）与对外“精准人格评估”承诺。
- 不做语音识别打分/口音纠错（可作为后续增值）。
- 不做重社交与 UGC（先把学习闭环和留存跑通）。

## 2. 用户画像与核心场景

### 2.1 学习者（6–12岁）
- 需要：短时高频、即时反馈、可获得成就感；不喜欢长时间输入与复杂规则。
- 行为：碎片化学习，常由家长引导；更愿意“玩”而不是“学”。

### 2.2 家长
- 需要：看得见效果（今日学了什么、是否能用在对话里）、轻陪伴、不需要额外准备材料。
- 行为：睡前/周末陪玩；希望产品能替代“我不知道怎么教英语”。

### 2.3 核心使用场景
- 家庭对话冲突高频：**想玩→家长忙→约时间→遵守规则**（CHAPTER 11-01/02/03）。
- 家庭小游戏：躲猫猫/剪刀石头布/捉迷藏/捉人（CHAPTER 11-04/05/06/07）。
- 安静活动：画画（CHAPTER 12-01）。

## 3. 核心机制：MBTI 趣味快测（MVP）

### 3.1 定位与对外话术
- 对外：**“MBTI + 学习性格/偏好”**（趣味测）。
- 对内：将用户映射为 **16 型 + 4 维学习偏好权重**，用于推荐玩法/反馈文案/节奏。

### 3.2 问卷形式（4 题，30 秒）
每题 2 选 1（A/B），配插画与情境，降低“考试感”。

- **Q1（E/I：互动偏好）**：你更想怎么学？
  - A：和人一起玩对话更有劲（E）
  - B：我先自己闯关，熟了再一起玩（I）
- **Q2（S/N：信息偏好）**：你更喜欢哪种提示？
  - A：给我例句，我照着用（S）
  - B：告诉我这句话在什么场景用（N）
- **Q3（T/F：反馈偏好）**：做错了你更想看到：
  - A：直接告诉我正确答案 + 规则（T）
  - B：先鼓励我，再告诉我更好的说法（F）
- **Q4（J/P：节奏偏好）**：你更喜欢的关卡：
  - A：有计时/有目标/拿星星（J）
  - B：可以自由选剧情/随机惊喜（P）

### 3.3 输出
- **MBTI 类型卡**：如 ENFP / ISTJ（用于传播与身份感）。
- **学习性格标签**：如“热场型/规则型/共情型/探索型”（更易理解）。
- **推荐策略**：给出“最适合你的 1 个场景 + 1 个玩法”一键开玩。

## 4. 内容与关卡（基于 `part2.json`）

### 4.1 内容来源与范围
- 句库：`part2.json`
- MVP 章节优先级：
  1) CHAPTER 11-01/02/03（强对话、强冲突、可做闯关）
  2) CHAPTER 11-04/05/06/07（小游戏词汇与规则）
  3) CHAPTER 12-01（画画：描述/请求/鼓励）

### 4.2 “对话里会用”的定义（学习产出）
用户在一个情境中能完成以下任一动作，即视为“会用”：
- **选对回应**：在 2–4 个句子中选出最合适的一句。
- **补全对话**：在对话缺口中选出下一句。
- **意图匹配**：把句子拖到“拒绝/约定/鼓励/规则”等意图标签下（MVP 可用选择题替代拖拽）。

## 5. 游戏玩法模板（MVP 必做 3 类）

### 模板 A：情境选择（核心）
- 输入：一段 2–4 轮对话 + 当前缺口意图
- 操作：从 2–4 个英文句子中选 1 个
- 反馈：正确/不合适原因（场景匹配解释）
- 适配：S/T/J 强；也可给 F 用“更温柔版本”的解释

### 模板 B：对话拼图（从句子到对话）
- 输入：一段对话被打散成 3–5 句
- 操作：选择下一句/排序（MVP 先做“选择下一句”）
- 目标：让用户理解句子之间的“轮次逻辑”

### 模板 C：道具卡（规则/时间/情绪卡）
用于把“家庭规则”可视化、降低抽象度：
- **时间卡**：5 分钟/20 分钟/1 小时
- **规则卡**：promise / timer / follow the rules
- **情绪卡**：bored / want to play now

（注：虽然 MVP 不做语音，但仍支持点读发音，形成听觉记忆。）

## 6. 个性化推荐（MBTI→玩法与节奏）

### 6.1 维度到产品策略的映射
- **E**：优先推“亲子双人对话”与“更强剧情演出”
- **I**：优先推“单人闯关”“更少干扰的界面”
- **S**：优先推“例句先行”“短回合高重复”
- **N**：优先推“场景解释先行”“多分支剧情”
- **T**：反馈更直接：对/错 + 规则点
- **F**：反馈更温柔：先鼓励 + 更好表达
- **J**：偏好“计时/清单/连续天数”
- **P**：偏好“随机事件/惊喜掉落/自由选关”

### 6.2 MVP 推荐规则（可解释）
优先级 = 场景热度（固定） + 类型匹配加权 + 错题相关度（近期）
- 场景热度：11-01/02/03 > 11-04/05/06/07 > 12-01
- 类型匹配：
  - J：优先 11-03（约定时间）
  - F：优先 11-01（邀请/安抚）
  - T：优先 11-05（规则明确）
  - P：优先 11-04/06（更“玩”的场景）

## 7. 功能列表（MVP）

### 7.1 必做
- MBTI 趣味快测（4 题）+ 类型结果页
- 今日推荐场景（1 个）+ 继续上次
- 场景关卡（模板 A + B + C）
- 点读发音（英文）
- 结算页：掌握句子、错题句子、继续下一关
- 收藏句子（用于复习）
- 订阅与创始会员入口（支付/权益展示）

### 7.2 可选（MVP+）
- 亲子双人模式（同屏轮流选句）
- 复习计划（间隔重复：1/3/7 天）
- 更强剧情分支与角色皮肤

## 8. 商业化：订阅 + 创始会员 0 元

### 8.1 订阅建议
订阅售卖的不是“内容本身”，而是“体系化对话训练 + 个性化推荐 + 可视化成长”：
- 解锁全部章节/场景包
- 每日推荐与复习计划
- 家庭多账号（后续）

### 8.2 创始会员（0 元加入）定位
目标：降低首次付费门槛，收集种子用户与使用数据，形成社群与口碑。

建议权益（不与付费订阅正面冲突）：
- 创始会员徽章 + 专属昵称边框（身份）
- 早鸟共创：投票决定下一章优先级
- 每周 1 个创始专属场景（小而精）
- 订阅折扣锁定（如首月/首年折扣；具体需与你方定价策略同步）

建议限制：
- 0 元加入需完成 1 次快测 + 1 个场景（保证质量线索）
- 创始名额/截止时间（制造稀缺与转化）

## 9. 数据指标与验收（MVP）

### 9.1 核心漏斗
- 进入→完成快测→开始第 1 关→完成第 1 关→进入订阅页→下单

### 9.2 学习效果（代理指标）
- 每关平均互动次数（点击/选择/点读）
- 错题率下降（同一意图题在 24h 内复做）
- “对话完成率”（对话拼图关卡完成）

### 9.3 体验指标
- 首局完成率
- 首局时长（目标 60–180 秒）
- 次日留存 D1（目标先以提升为导向，不设硬 KPI）

## 10. 里程碑（建议 3–4 周可上线）
- Week 1：内容结构化（11-01/02/03）+ 模板 A + 支付/创始会员占位
- Week 2：模板 B + 收藏/结算/今日推荐
- Week 3：11-04/05/06/07 上线 + 类型推荐权重 + 埋点
- Week 4：增长实验（创始会员活动）+ 首轮迭代
