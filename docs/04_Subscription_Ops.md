# 订阅策略与运营方案（MVP：订阅 + 0 元创始会员）

## 1. 商业目标与原则
- **目标**：从一开始就以订阅为主线，同时用“0 元创始会员”完成冷启动（种子用户 + 口碑 + 数据）。
- **原则**
  - 创始会员不与订阅正面冲突：给身份与共创权益，订阅给系统性内容与训练体系。
  - 对亲子友好：少打扰、可陪伴、看得见成长。
  - H5 转化：路径短、权益清晰、退款与续费规则清楚。

## 2. 0 元创始会员（Founder Pass）

### 2.1 定位
“加入共创的早期用户身份”，用于建立信任与社群，降低首次下单门槛。

### 2.2 加入条件（建议）
- 完成：MBTI 快测 + 1 个场景关卡
- 领取：创始会员徽章（可生成分享卡）

### 2.3 权益设计（建议）
不抢订阅核心价值，但足够“想要”：
- 创始徽章 + 专属头像框/昵称边框（身份）
- 每周 1 个创始专属小场景（限量内容，避免大规模侵蚀订阅）
- 共创投票：下一批场景优先级/插画风格投票
- 订阅早鸟权益：例如首月折扣/首年折扣锁定（与你方定价策略同步）

### 2.4 限制与节奏
- 有截止日期/名额（稀缺）
- 0 元加入默认不自动续费（建议）

## 3. 订阅产品结构（建议）

### 3.1 订阅售卖的“核心价值”
从“句子库”升级为“对话训练系统”：
- 解锁全部场景包（持续更新）
- 个性化推荐（MBTI/偏好 + 错题复习）
- 复习计划（间隔重复，MVP 可先简化为“错题再练 + 收藏复习”）
- 家庭模式/多档案（后续）

### 3.2 价格与周期（文档占位）
H5 早期建议提供：
- 月订阅
- 年订阅（更优价）
（具体数值由市场与投放测算决定。）

## 4. 增长与留存（MVP 可执行）

### 4.1 冷启动（第 1–2 周）
目标：让用户快速体验“个性化 + 可用在对话里”的差异。
- 首次进入必推：快测 → 推荐场景（11-01/02/03）
- 结算页强引导：收藏 3 句 + 再来 1 关
- 创始会员弹层：只在完成第 1 关后出现（避免打断）

### 4.2 留存活动（第 3–4 周）
目标：形成“每天 3 分钟”的习惯。
- **7 日对话挑战**：每天一个家庭小场景（推迟/约时间/规则/鼓励）
- **连胜奖励**：连续 3/7 天送“角色贴纸/边框”（不影响订阅价值）
- **亲子任务**：同屏轮流答 5 题（后续可做）

### 4.3 分享与裂变（轻量）
适合亲子传播的内容：
- **类型卡分享**：MBTI 卡 + 学习性格标签 + “我最适合的场景”
- **成长战报**：本周掌握了 X 句“家庭口语”
- **挑战海报**：7 日挑战第 N 天完成

## 5. 运营内容日历（建议）
以“场景包”为运营单位：
- Week A：想玩/无聊（11-01）
- Week B：下次再玩（11-02）
- Week C：约定时间（11-03）
- Week D：躲猫猫/捉迷藏（11-04/06）
- Week E：剪刀石头布规则王（11-05）
- Week F：画画描述（12-01）

## 6. 订阅转化策略（H5）

### 6.1 触发点（MVP）
- 完成第 1 关结算页：提示“解锁更多场景包”
- 用户收藏达到 5 句：提示“开启复习计划（订阅）”
- 连续学习第 2 天：提示“订阅让推荐更准 + 解锁更多关”

### 6.2 权益对比建议
- **免费/创始会员**：体验少量关卡 + 身份/共创
- **订阅**：全部章节 + 个性化推荐 + 复习体系 + 未来家庭多档案

## 7. 风控与口碑
- 避免夸大“MBTI 精准”：强调“趣味测/学习偏好”
- 对儿童体验：不做“红叉羞辱”，反馈温和
- 付费透明：明确是否自动续费、取消入口
