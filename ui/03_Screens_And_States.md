# 页面线框（文字版）+ 状态清单（给前端工程师 Agent）

> 对齐：`docs/02_UI_UX.md` 的 P0–P7  
> 目标：不等设计稿也能把页面结构和状态做对

## P0 `/` 落地页
### 布局
- 顶部：Logo/标题
- 中部：一句话定位 + 3 个亮点（对话、亲子、个性化）
- 底部：CTA（开始测试 / 直接开玩）

### 状态
- loading（可选）
- 正常

## P1 `/quiz` 趣味快测
### 布局
- 顶部：进度条 `1/4`
- 中部：问题卡（标题 + 插画占位）
- 底部：A/B 两个大按钮

### 状态
- 初始题
- 进行中（已答 n 题）
- 完成跳转（loading 200ms）

## P2 `/result` 结果页
### 布局
- 类型卡（ENFP）
- 学习性格标签（Chip x2）
- 推荐理由（3 条）
- CTA：立即开玩

### 状态
- 正常
- 无结果（fallback：提示先测 / 跳 `/quiz`）

## P3 `/home` 首页
### 布局
- 今日推荐卡（场景名/目标/预计时长/开始）
- 继续上次（若有）
- 功能入口：订阅/创始会员、收藏

### 状态
- 无测评：显示“先测一下更准”按钮
- 有测评：显示推荐
- 内容加载失败：提示 + 重试

## P4 `/level/:levelId` 关卡页
### 布局
- 顶部：返回 + 题目进度
- 中部：对话气泡区（context / dialogue_so_far）
- 下部：选项区（2–4）+ 点读提示
- 底部：反馈层出现后显示“继续”

### 状态
- loading：骨架屏
- answering：可选项
- answered：锁选项 + FeedbackPanel + Next
- completed：跳转 summary
- audio_not_supported：toast，不阻塞

## P5 `/summary/:levelId` 结算页
### 布局
- 成绩区：正确数/星级
- 掌握句子列表（可点读/收藏）
- 错题列表（可再练）
- CTA：下一关 / 去订阅 / 回首页

### 状态
- 正常
- 无错题：隐藏错题模块

## P6 `/library` 收藏
### 布局
- 列表：SentenceCard（点读/取消收藏）
- CTA：开始复习（进入复习关卡）

### 状态
- 空态：提示“先去关卡里收藏几句”

## P7 `/paywall` 订阅/创始会员
### 布局
- FounderCard：0 元加入（权益+条件）
- SubscribeCard：订阅（权益+价格占位）
- FAQ：自动续费/取消

### 状态
- 加入中/成功
- 支付中/成功/失败（toast + 重试）
